Time tag & WooCommerce allowed countries and states tags for Contact Form 7.

== Description ==

Add custom tag:
• WooCommerce allowed countries dropdown field: please add 'city_container' class on city field container and 'zip_container' class on zip code field container, to update fields depending on country data returned.
• Dynamic States/Provinces dropdown field: please 'state_container' class on your States/Provinces field container, to update field depending on country data returned.
• Time input field, with min and max options

= Notice : =
Countries influence on State title and ZIP code & city fields. To update these fields :
-- Please add 'state_container' class on state field container,
-- Please add 'city_container' class on city field container,
-- Please add 'zip_container' class on zip code field container.

= Translations =

The plugin is translated into Arabic, German, Spanish, French, Italian and Portuguese.

== Installation ==

1. Upload `cf7-twx-custom-tags` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. This plugin need WooCommerce and Contact Form 7 installed
